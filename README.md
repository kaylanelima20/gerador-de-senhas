# gerador-de-senhas
 simples gerador de senhas para praticar Javascript.

## Funcionalidades
- gerar senhas aletórias
- escolher caracteres da senha
- definir o tamanho da senha
- exibir senha para o usuário

## Tecnologias
- HTML
- CSS
- JavaScript

## Aprendizados
- math.random()
- manipulação de elementos
- laços de repetição
